# 🎵 Хөгжмийн Ертөнц

Энэхүү вэбсайт нь хүүхдүүдэд хөгжмийн нот, тэмдэг, тэмдэглэгээнүүдийг уйтгартай биш, хаанаас ч хамаагүй хамгийн сонирхолтой бөгөөд хөгжилтэй хэлбэрээр сурахад нь туслах зорилготой интерактив вэб хуудас юм.

## 🎯 Төслийн зорилго
Хүүхдүүд хөгжмийн онолыг тоглоом харах мэт сонирхож, өөрийн хүссэн газраас, дуртай төхөөрөмжөөрөө (утас, компьютер) дамжуулан хялбархан суралцах боломжийг бүрдүүлэх.

## 💻 Компьютер дээрээ ажиллуулах заавар
Энэхүү төсөл нь одоогоор хөгжүүлэлтийн шатанд байгаа тул дараах алхмаар өөрийн компьютер дээрээ ажиллуулж үзнэ үү:

1. Баруун дээд буланд байх **Code** товчлуур дээр дараад **Download ZIP** сонголтоор кодыг татаж авна.
2. Татсан `.zip` файлаа задална (Extract).
3. Хавтас доторх **`index.html`** файл дээр хулганы баруун товчийг дараад, дурын браузер (Chrome, Safari гэх мэт) ашиглан нээнэ.

## ✨ Онцлогууд
* **Хүүхдэд ээлтэй дизайн:** Өнгөлөг, харахад урамтай өнгө төрх.
* **Хаанаас ч хандах боломж:** Утас, таблет, компьютер дээр зэрэг ажиллана.
* **Интерактив:** Нотууд болон хөгжмийн тэмдэглэгээнүүд дээр дарахад сонирхолтой хэлбэрээр суралцах боломжтой.

## 🛠️ Ашигласан технологиуд
* HTML5 (Бүтэц)
* CSS3 (Дизайн болон анимаци)
* JavaScript (Интерактив үйлдэл)
